# Petualangan Raki Membuat Code Snippets

```
Disclaimer: Kisah ini fiktif.
```

"Source code aplikasi yang besar terlalu panjang...", kata Raki dalam hati.

Malam itu Raki sedang bosan, entah apa penyebabnya.

Sambil bersandar, Raki mencoba melakukan brainstorming tentang kegiatannya di malam itu.

30 menit berlalu, belum ada ide, Raki memutuskan untuk browsing sejenak.

Satu, dua, tiga domain terlampaui.

Hingga pada akhirnya Raki menemukan repository yang berisi kode pendek, tapi cukup berguna.

"Code snippets!", kata Raki dalam hati.

Mengambil langkah cepat, Raki memutuskan untuk membuat repository baru dan melakukan:

```
git clone git@example.com:raki/code-snippets.git

cd code-snippets

code .
```

Walaupun kecil, code snippets cukup powerful untuk mengerjakan task yang spesifik.

Tak lama kemudian, salah satu file berisi code snippets selesai.

Setelah menyelesaikan kode ini:

```
// snippet-0.js: console.log example

console.log("old macdonald had a code, 0x313100 ...")
````

Rocky menuju terminal dan melakukan:

```
git add .

git commit -m "0x313100"

git push -u origin main
```

Tiba-tiba saja source code snippets Raki terupload dan tugas Raki malam itu selesai.
